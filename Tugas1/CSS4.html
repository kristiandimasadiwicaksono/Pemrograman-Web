PRAKTIKUM PEMROGRAMAN WEB

Pemrograman Web memberikan Ketrampilan kepada mahasiswa dalam pembuatan situs web yang dinamis dan dapat berinteraksi dengan user. Materi meliputi: Teknologi Web, Pembuatan Situs, HTML, CSS, PHP, MySQL dan Visual Studio Code. "E-Learning" link.

Keterangan:

  1. Warna border adalah yellowgreen
  2. Gunakan transformasi teks uppercase pada heading
  3. Paragraf utama menggunakan indentasi dan rata kiri
  4. Tabel header rata tengah
  5. Baris ganjil rata kiri, baris genap rata kanan kecuali pada kolom icon, gunakan fitur nth-child
